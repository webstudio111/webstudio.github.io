# 🏭 Front Desk Kit

A complete, zero-cost starter kit for selling **AI receptionists to local businesses** — the highest-probability AI side business of 2026 per actual earnings data (not guru hype).

**The pitch you'll be making:** small businesses lose ~$126K/year to missed calls. You install an AI assistant that answers instantly and books appointments 24/7. It costs you $0–99/month to run and businesses pay $99–299/month + setup fees for it.

**Your unfair advantage:** the Demo Factory. Personalized demos close ~4× better than generic pitches — and this kit makes one in 2 minutes.

## Quick start

1. Enable **GitHub Pages** on this repo (Settings → Pages → deploy from branch → root)
2. Open `front-desk-kit/index.html` on your Pages URL
3. Read `PLAYBOOK.md` — then go send 10 demos

Everything runs in the browser. No server, no database, no dependencies, no costs. Demo configurations are encoded directly in the shareable links.
